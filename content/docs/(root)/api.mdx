## Components API

Description of props of each component.

## Map

The `Map` component is the main map container. It wraps `react-leaflet`’s `MapContainer` and provides default styles.

### Props

| Name           | Type                | Description                                                                                    |
| -------------- | ------------------- | ---------------------------------------------------------------------------------------------- |
| `center`       | `LatLngExpression`  | **Required.** The initial center of the map `[lat, lng]`.                                      |
| ...other props | `MapContainerProps` | All other props from [React Leaflet MapContainer](https://react-leaflet.js.org/docs/api-map/). |

## MapTileLayer

The `MapTileLayer` component adds a tile layer to the map. **It is required for the map to be rendered.** It automatically switches between light and dark map styles based on the current theme from `next-themes`. You can also provide a custom `url` or `attribution` if needed.

### Props

| Name           | Type             | Description                                                                                                  |
| -------------- | ---------------- | ------------------------------------------------------------------------------------------------------------ |
| `url`          | `string`         | Optional. URL of the tile server. Defaults to Carto’s light/dark tiles.                                      |
| `attribution`  | `string`         | Optional. HTML string for map attribution. Defaults to OpenStreetMap and CARTO attribution.                  |
| ...other props | `TileLayerProps` | All other props from [React Leaflet TileLayer](https://react-leaflet.js.org/docs/api-components/#tilelayer). |

## MapDefaultMarkerIcon

The `MapDefaultMarkerIcon` component renders the default marker icon used by `MapMarker`. Use this component if you want to customize the appearance of the default icon.

### Props

| Name     | Type          | Description                                  |
| -------- | ------------- | -------------------------------------------- |
| ...props | `LucideProps` | Any props supported by `lucide-react` icons. |

## MapMarker

The `MapMarker` component renders a marker on the map. By default, it uses `MapDefaultMarkerIcon`, but you can provide a custom React node as the `icon` to override it. It supports customization of the icon’s position, popups, and tooltips using standard Leaflet `DivIcon` [options](https://leafletjs.com/reference.html#divicon).

### Props

| Name            | Type                        | Description                                                                                                |
| --------------- | --------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `icon`          | `ReactNode`                 | Optional. Custom icon to display instead of the default marker. Defaults to `MapDefaultMarkerIcon`.        |
| `bgPos`         | `[number, number]`          | Optional. Background position offset for the icon.                                                         |
| `iconAnchor`    | `[number, number]`          | Optional. The point of the icon which will correspond to marker’s location.                                |
| `popupAnchor`   | `[number, number]`          | Optional. Point from which the popup should open relative to the icon.                                     |
| `tooltipAnchor` | `[number, number]`          | Optional. Point from which the tooltip should open relative to the icon.                                   |
| ...other props  | `Omit<MarkerProps, "icon">` | All other props from [React Leaflet Marker](https://react-leaflet.js.org/docs/api-marker/), except `icon`. |

## MapPopup

The `MapPopup` component renders a popup. It provides default styling and animations that match the shadcn/ui design system.

### Props

| Name     | Type         | Description                                                                                    |
| -------- | ------------ | ---------------------------------------------------------------------------------------------- |
| ...props | `PopupProps` | All props from [React Leaflet Popup](https://react-leaflet.js.org/docs/api-components/#popup). |

## MapTooltip

The `MapTooltip` component renders a tooltip. It provides default styling and animations that match the shadcn/ui design system.

### Props

| Name        | Type                                     | Description                                                                                        |
| ----------- | ---------------------------------------- | -------------------------------------------------------------------------------------------------- |
| `direction` | `"top" \| "right" \| "bottom" \| "left"` | Controls the position of the tooltip relative to the target element. Defaults to `"top"`.          |
| ...props    | `TooltipProps`                           | All props from [React Leaflet Tooltip](https://react-leaflet.js.org/docs/api-components/#tooltip). |
